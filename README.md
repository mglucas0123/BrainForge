# BrainForge

Portable agent kit (caveman + cavemem + RTK + skills) for **Cursor**, **GitHub Copilot**, and **Antigravity**.

## Quick start

```powershell
# Build CLI (Rust)
cargo build --release

# Sync to current project (interactive IDE picker)
.\target\release\brainforge.exe sync

# Or non-interactive
.\target\release\brainforge.exe sync --adapter all --no-menu

# Health check
.\target\release\brainforge.exe doctor

# Memory (canonical: brainforge/memory/)
.\target\release\brainforge.exe memory read
.\target\release\brainforge.exe memory refresh    # realign **Capacity:** only
.\target\release\brainforge.exe memory compress   # heuristic; use /compress-context in chat for semantic
.\target\release\brainforge.exe memory validate  # CI: exit 1 on errors
```

## Layout

| Path | Role |
|------|------|
| `brainforge/` | Kit source (core, memory, adapters, RTK) |
| `crates/brainforge-core/` | Sync + doctor library |
| `crates/brainforge-cli/` | `brainforge` binary |
| `.cursor/`, `.github/`, `.agents/` | Generated by `brainforge sync` |

Edit **`brainforge/`** → run **`brainforge sync`**.  
Memory: **`brainforge/memory/`** (per project; clear `.context.md` when copying kit).

## Copy to another project

1. Copy folder **`brainforge/`** (include `tools/rtk/rtk.exe` or run `install-rtk-local.ps1`).
2. Copy `target/release/brainforge.exe` or install via `cargo install --path crates/brainforge-cli`.
3. `brainforge sync` in the host repo.

See `brainforge/HOST-SETUP.md`.
